Destaques no texto

Podemos realizar destaques no texto, tais como:
negrito utilizando as tags < b > ... < /b >
texto destacado utilizando as tags < strong > ... < /strong >
itálico utilizando as tags < i > ... < /i >
texto com ênfase utilizando as tags < em > ... < /em >

A diferença entre a tag < b > que faz negrito para a tag < strong > é que a tag strong dará um destaque semântico o mesmo é valido para a tag < em >
